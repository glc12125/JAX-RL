# JAX-RL
JAX implementations of various reinforcement learning algorithms.

Main libraries used:
* JAX - main framework
* Haiku - neural networks in JAX
* optax - gradient based optimisation in JAX

Algorithms:
* Deep Q-Network (DQN)

Policies:
* Epsilon-greedy
* Boltzmann
