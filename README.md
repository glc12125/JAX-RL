# JAX-RL
JAX implementations of various deep reinforcement learning algorithms.

Main libraries used:
* JAX - main framework
* Haiku - neural networks
* Optax - gradient based optimisation

Algorithms:
* Proximal Policy Optimization
* Deep Recurrent Q-Network (DRQN)

Policies:
* Epsilon-greedy
* Boltzmann
